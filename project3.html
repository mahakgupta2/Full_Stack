Amazon Clone

Objectives


1. Cross-Platform Music Experience

- Build a modern, responsive music player for Android, iOS, Web, Windows.
- Ensure smooth playback with minimal lag and elegant transitions.

2. User-Friendly Interface

- Intuitive navigation via BottomNavigationBar (Home, Search, Favorites, Music, Profile).
- Attractive UI with glowing buttons, dark theme, and soft animations.

3. Role-Based Access

- Secure Firebase Authentication for login/signup
. - Role-based redirection: Admin → /adminDashboard | User → /userHome

4. Streaming & Playback

- Stream songs directly from Cloudinary URLs.
- Provide all key audio controls: Play/Pause, Next/Previous, Seekbar.

5. Personalization

- Favorites management for songs.
- Playback state persistence with HydratedBloc.

Deliverables


1.Music Playback System

- Core player using audioplayers package.
- Persistent bottom controls for quick play/pause.

2.Search & Navigation

- Search songs by title/artist with real-time filtering.
- “See All” option for complete song lists.

3.Favorites Module

- Add/remove songs to Favorites.
- Dedicated Favorites Tab.

4.Profile & Settings

- Display user email, profile options, and logout button with glowing effect.

5.Admin Functionality

- Separate Admin Dashboard for managing content.

Requirements

Requirement Details
Framework Flutter SDK (cross-platform).
Backend Firebase Authentication for login, signup, role-based access.
Cloud Storage Cloudinary for hosting and streaming audio files.
Audio Package audioplayers / just_audio for playback functionality
State Management HydratedBloc for persisting UI and playback state.
UI/UX Material Design, glowing buttons, circular images,animations
